# Flappy Bird Qt

基于 Qt6 开发的 Flappy Bird 游戏。

## 版本历史

- **v3.0** - 皮肤选择系统：蓝/黄小鸟可选，完整8帧动画
- **v2.0** - 新增移动管道系统，管道可上下往复移动
- **v1.0** - 初始版本，基础游戏玩法

## 创新点

### 1. 移动管道系统
- 管道可沿垂直方向往复移动（最大移动距离 50px），分为普通移动管道和悬浮移动管道两种类型
- 动态难度调整：分数越高，移动管道出现的概率越大

### 2. 皮肤选择系统
- 开始界面可通过方向键切换蓝/黄小鸟
- **蓝色小鸟**：8帧完整飞行动画 + 8帧下落动画 + up.png + dead.png，共18张独立图片
- **黄色小鸟**：3帧简化循环动画，轻巧灵动
- 未选中皮肤显示暗淡效果，选中皮肤带动画

### 3. 多态管道架构
- 采用 Pipe 基类 + MovingPipe 子类的继承体系
- 利用 Qt 虚函数实现运行时多态，游戏元素层无需关心具体管道类型

## 优点

### 视觉效果
- 30 FPS 流畅游戏循环
- 小鸟翅膀动画（蓝色8帧完整循环）
- 背景无限滚动，层次感强
- 云朵飘动效果（含随机缩放 1.0~2.0 倍）
- 三种管道类型：普通管道、悬浮管道、移动管道
- 游戏欢迎界面与结束界面（带闪烁动画）
- 角色皮肤选择预览动画

### 音效系统
- 飞行音效：轻盈的扑翅声
- 碰撞音效：真实的撞击声
- 得分音效：清脆的得分反馈

### 技术特性
- **对象池模式**：管道复用，避免频繁内存分配，提升性能
- **单例模式**：计分器、皮肤管理器全局唯一
- **虚函数多态**：Pipe 基类 + MovingPipe 子类，代码优雅可扩展
- **资源压缩**：Qt 资源系统 (.qrc) 打包所有素材

## 如何编译运行

### 方法一：下载源码自行编译

**环境要求**
- Qt 6.x
- MinGW 64-bit 编译器

**编译步骤**

```bash
# 1. 克隆仓库
git clone https://github.com/bravver/c++_flappybird.git
cd c++_flappybird

# 2. 使用 Qt Creator 打开项目
# 打开 flappybird-qt.pro 文件

# 3. 在 Qt Creator 中点击"运行"即可编译并运行
```

### 方法二：下载预编译版本

前往 [Releases](https://github.com/bravver/c++_flappybird/releases) 页面下载最新版本的 `flappybird-qt.exe`，双击即可运行。

## 游戏规则

- 点击空格键让小鸟上升
- 小鸟会在重力作用下自然下落
- 避开所有管道，存活越久分数越高
- 碰撞管道或地面则游戏结束
- 最高分会自动保存
- 使用方向键选择喜欢的小鸟皮肤

## 项目结构

```

├── main.cpp                    # 程序入口
├── game.h / game.cpp           # 游戏主窗口
├── constant.h / constant.cpp   # 常量定义
├── bird.h / bird.cpp           # 小鸟类
├── pipe.h / pipe.cpp           # 水管基类
├── movingpipe.h / cpp          # 移动水管类
├── pipepool.h / pipepool.cpp   # 水管对象池
├── gamebackground.h/cpp        # 游戏背景
├── gameforeground.h/cpp        # 游戏前景(云朵)
├── gameelementlayer.h/cpp     # 游戏元素层
├── scorecounter.h/cpp         # 计分器
├── skinmanager.h/cpp           # 皮肤管理器(v3.0新增)
├── welcomanimation.h/cpp      # 欢迎动画
├── gameoveranimation.h/cpp     # 结束动画
└── resources/                  # 图片和音效资源
    ├── img/                    # 蓝色小鸟图片资源
    ├── skin/yellow/            # 黄色小鸟图片资源(v3.0新增)
    ├── wav/                    # 音效资源
    └── readme_img/             # README截图
```
